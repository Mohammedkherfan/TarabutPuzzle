# TarabutPuzzle

### Language and Framework

* [JAVA] version 8
* [MAVEN] version 3.2.0


## Instalation

#### Build

```
mvn clean install
```


#### Jar

```
java -jar TarabutPuzzle-1.0-SNAPSHOT.jar
```


#### TDD

```
mvn install
```


#### Compilation

```
mvn install -DskipTests
```


#### Main class

Main class is `com.tarabut.puzzle.LaunchApplication` class
